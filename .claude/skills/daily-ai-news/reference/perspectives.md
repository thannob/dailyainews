# Perspectives — 2026-05-21

## 1. Anthropic จ่าย xAI เดือนละ 1.25 พันล้านดอลลาร์ค่า compute (TechCrunch)

**อาจารย์ (มหาวิทยาลัย):** ดีลนี้เป็นกรณีศึกษาว่า "ต้นทุนของ AI" ส่วนใหญ่ไม่ใช่ค่าอัลกอริทึม แต่เป็นค่าศูนย์ข้อมูลและ GPU — และที่น่าสนใจคือคู่แข่งโดยตรงสองค่ายยังต้องพึ่งโครงสร้างพื้นฐานของกันและกัน เปิดประเด็นสอนเรื่อง economics ของ compute และการพึ่งพากันในอุตสาหกรรม
**ผู้เชี่ยวชาญด้าน AI:** ก่อนดีลนี้ Colossus 1 ถูกใช้งานเพียงราว 11% — การที่ Anthropic เข้าเช่ากำลังประมวลผลที่ว่างอยู่สะท้อนว่า GPU idle คือสินทรัพย์ที่แพงเกินกว่าจะปล่อยทิ้ง ตลาด compute กำลังกลายเป็นตลาดซื้อขายล่วงหน้าแบบ commodity ที่มีสัญญาระยะยาว
**โปรแกรมเมอร์มืออาชีพ:** ดีลนี้ไม่กระทบโค้ดวันนี้ แต่เป็นสัญญาณว่ากำลังประมวลผลสำหรับเทรนโมเดลถูกจองล่วงหน้าถึงปี 2029 — capacity planning ระดับนี้แปลว่าราคา API ในอนาคตจะผูกกับสัญญา compute เหล่านี้ คุ้มที่จะติดตามเพราะมันกระทบต้นทุนต่อ token ที่เราจ่าย

## 2. Intuit ปลดพนักงานกว่า 3,000 คน หันโฟกัสไปที่ AI (TechCrunch)

**อาจารย์ (มหาวิทยาลัย):** เคสนี้สอนเรื่อง "AI กับตลาดแรงงาน" ได้ตรงประเด็นและเป็นรูปธรรม — บริษัทที่กำไรเพิ่มขึ้นยังปลดคน 17% โดยอ้างการปรับโฟกัสไปที่ AI ควรชวนนักศึกษาวิเคราะห์ว่าอะไรคือเหตุผลเชิงโครงสร้างจริง อะไรคือการสื่อสารต่อตลาดทุน
**ผู้เชี่ยวชาญด้าน AI:** จุดที่ต้องอ่านระวังคือ Intuit ทำสัญญาหลายปีกับทั้ง Anthropic และ OpenAI เพื่อฝังโมเดลในผลิตภัณฑ์ — "ปลดคนเพราะ AI" จึงไม่ได้แปลว่า AI ทำงานแทนคนได้ครบ แต่เป็นการจัดสรรงบจากค่าจ้างไปเป็นค่า licensing โมเดลและการลงทุน AI
**โปรแกรมเมอร์มืออาชีพ:** การที่เครื่องมือบัญชี/ภาษีของ Intuit จะไปปรากฏใน Claude และ ChatGPT บอกทิศทางชัดว่า surface ของซอฟต์แวร์กำลังย้ายเข้าไปอยู่ในแชท — ถ้าคุณสร้างผลิตภัณฑ์ B2B การคิดเรื่อง integration กับ assistant platform เริ่มเป็นเรื่องของกลยุทธ์ ไม่ใช่ของเสริม

## 3. Stability AI ออกโมเดลเสียงใหม่ สร้างเพลงยาว 6 นาทีได้ (TechCrunch)

**อาจารย์ (มหาวิทยาลัย):** การกระโดดจากเพลงสั้นใน Stable Audio 2.0 มาเป็นเพลงเต็มราว 6 นาที เป็นตัวอย่างที่ดีของการสอนว่า "ความยาวของ context ที่สอดคล้องกัน" คือโจทย์ยากของ generative model ไม่ว่าจะข้อความ เสียง หรือวิดีโอ — โครงสร้างเพลงที่ต่อเนื่องคือการทดสอบ long-range coherence
**ผู้เชี่ยวชาญด้าน AI:** จุดที่น่าสนใจเชิงเทคนิคคือ Stability เคลมว่าโมเดลเทรนบนข้อมูลที่มีลิขสิทธิ์ครบ และมีรุ่นเล็กที่รันบนเครื่องได้ — เป็นการตอบโจทย์สองด้านพร้อมกัน: ความเสี่ยงทางกฎหมายเรื่องข้อมูลเทรน และความต้องการ inference แบบ on-device ที่ไม่ต้องส่งข้อมูลขึ้นคลาวด์
**โปรแกรมเมอร์มืออาชีพ:** โครงสร้างไลน์อัป (Small/Medium/Large) บวกเงื่อนไข enterprise license สำหรับบริษัทรายได้เกิน 1 ล้านดอลลาร์ เป็นเครื่องเตือนให้อ่าน license ก่อน integrate — โมเดล open-weight ไม่ได้แปลว่าใช้เชิงพาณิชย์ได้ฟรีเสมอ ตรวจเงื่อนไขให้ตรงกับสเกลของผลิตภัณฑ์

## 4. Figma เพิ่ม AI agent ลงบน collaborative canvas (TechCrunch)

**อาจารย์ (มหาวิทยาลัย):** แนวคิดที่ Figma วาง AI ให้ "นั่งทำงานบน canvas เดียวกับคน" แทนที่จะเป็นเครื่องมือข้างๆ workflow เป็นตัวอย่างดีของการออกแบบ human-AI collaboration — UX ของ AI ไม่จำเป็นต้องเป็นแค่กล่องแชท แต่คือการวาง agent ไว้ในบริบทของงานจริง
**ผู้เชี่ยวชาญด้าน AI:** การที่ Figma "ทำให้ตัว Figma เองอ่านได้โดยโมเดล" — ให้โมเดลเข้าใจ component, token, มาตรฐาน — คือหัวใจ ความสามารถ agentic ที่ดีไม่ได้มาจากโมเดลใหญ่อย่างเดียว แต่มาจากการทำให้ domain ของงานเป็นข้อมูลที่ machine-readable
**โปรแกรมเมอร์มืออาชีพ:** ฟีเจอร์รัน agent หลายตัวพร้อมกันเพื่อทำงานคู่ขนานเป็นรูปแบบที่เริ่มเห็นซ้ำในเครื่องมือ dev — และตอนนี้ฟรีช่วง beta แต่จะกลายเป็น usage-based paid ภายหลัง ทดลองได้ แต่ควรเผื่อ budget ไว้ก่อนพึ่งมันใน production pipeline

## 5. สตาร์ทอัพ AI search กำลังบูม — Exa Labs ระดมทุน 250 ล้านดอลลาร์ (TechCrunch)

**อาจารย์ (มหาวิทยาลัย):** การที่ valuation ของ Exa เพิ่มกว่าสามเท่าในเวลาไม่ถึงปี เป็นกรณีศึกษาเรื่อง venture economics ในยุค AI — และสอนแนวคิดสำคัญว่า "search สำหรับ AI" ต่างจาก "search สำหรับมนุษย์" เพราะผู้บริโภคผลลัพธ์คือโมเดล ไม่ใช่คน
**ผู้เชี่ยวชาญด้าน AI:** โจทย์ที่ Exa จับคือการป้อนข้อมูลสดจากเว็บให้ LLM อย่างแม่นยำ — นี่คือชั้น retrieval ที่อยู่เบื้องหลังคุณภาพของ RAG และ agent ทั้งหลาย การมีโครงสร้างพื้นฐาน search เฉพาะทางแยกออกมาเป็นสัญญาณว่าวงการมอง retrieval เป็นปัญหาที่ใหญ่พอจะมีบริษัทระดับพันล้านมาแก้
**โปรแกรมเมอร์มืออาชีพ:** ถ้าคุณสร้าง agent หรือระบบ RAG อยู่ การมีตัวเลือก search API เฉพาะ AI มากขึ้น (Exa เคลมรองรับนักพัຒนากว่า 400,000 คน) แปลว่ามีทางเลือกนอกเหนือจาก search ของ OpenAI/Google — คุ้มที่จะ benchmark คุณภาพ retrieval และราคาเทียบกันก่อนล็อกอินกับเจ้าใดเจ้าหนึ่ง
